# 序言

这本书写给一种特定的人：从杭州出发，不开车，出门希望舒适大于折腾，能用积分就用万豪积分，商务出行愿意多花一千买不累。

预设决定了取舍。杭州东是这本书的原点，所有线路、所有时长，都从这一站算起。不自驾意味着行程围着高铁站和市内公共交通转，景点能不能直达决定了它值不值得写。舒适优先意味着坐二等座撑五小时不在考虑范围，长途首选商务座或一等座，住宿优先连锁可控的万豪体系。

选城市用三条标准筛。第一条，景点本身经得起一次专程。第二条，万豪在当地有像样的店，至少是一家行政酒廊靠谱、用积分换房不亏的酒店。第三条，杭州东出发的高铁班次足够多，最好每小时都有车，避免被一趟车的时刻表绑死全天。三条都满足才进入推荐范围。

不写的内容也提前讲清。不写穷游路线，青旅、绿皮、十二小时硬座这本书一概不涉及。不写自驾，租车、停车场、川藏线山路都不在范围。不写民宿和短租，定不到、临时取消、周边吵这些坑这本书绕开。也不写所谓小众秘境，去了发现没维护或者不通车的地方不值得排进有限的假期。

剩下的就是正题。每一章先告诉你季节和时长，再讲怎么坐车到，最后讲住哪一家、看什么、吃什么。结构尽量短，决策尽量准。
