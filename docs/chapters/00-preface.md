# 序言

这本书写给一种相当具体的人。出发城市是杭州，平时不开车，出门希望舒适胜过折腾，能用积分的时候就动用万豪积分，对于一次商务行程或一段周末小假，愿意多花一千块钱换一段不那么累的旅途。这本书的整套取舍，都是从这种生活方式出发推回来的。

杭州东站是这本书的原点。所有线路的距离、所有车程的长短、所有酒店的位置选择，都是从这一站算起的。换句话说，本书把杭州东当作圆心，一圈一圈往外划。一个半小时之内能到的，划进短途圈；三到四个小时的，划进中途圈；五到七小时的归到长途圈；再往外，就需要借助转车、动卧或者跨夜列车的超长途圈。这种以杭州东为圆心的同心圆结构贯穿全书，所有章节的排序方式都是这一规则的延伸。

不自驾的预设决定了行程的边界。租车、停车、山路盘旋、自驾过夜，这些都不在讨论范围之内。每一座城市的取舍标准，都要看城市公共交通和景点之间的连接是否顺畅；如果一处景点只能依靠包车或自驾才能抵达，这本书会直接放弃。这种取舍的代价是少了一些深山秘境，换回来的好处是行程的可预期性。换言之，每一段路程都能用 12306 应用、滴滴打车和酒店礼宾排得明明白白，不必担心临时找不到司机或者错过班车。

舒适优先这一条预设进一步压缩了选择。一段五小时以上的高铁旅程，坐在二等座上其实可以坚持，但坚持之后下车的状态并不适合立刻投入新的城市。本书在长途档默认推荐一等座或商务座，价格虽然贵出一截，但抵达之后人的精神状态完全不一样。住宿层面，本书优先推荐万豪 Bonvoy 体系内的酒店。一方面是因为万豪在中国大陆的覆盖密度足够高，几乎每一座值得专程造访的城市都有至少一家拿得出手的店；另一方面是因为积分体系成熟，长期累积下来，住宿成本可以被显著压低。当一座城市的万豪存在感比较弱时，本书会在万豪推荐之外补一段"当地特色酒店"，把那些虽不在连锁体系之内、但在调性和位置上格外出色的选项一并列入。

筛城市用三条标准。第一条，这座城市本身要值得专程跑一趟，景点的厚度和密度要足以撑起至少一个完整的周末，否则便算不上目的地。第二条，万豪在这座城市要有像样的店：一家行政酒廊运转正常、用积分换房不会感觉吃亏的酒店，是这条标准的最低线；如果当地没有满足这条线的万豪，本书会写明替代选项，但不会勉强把一座没有像样酒店的城市排进来。第三条，从杭州东出发的高铁班次要足够多，最好做到每小时都有车，避免被某一班车的时刻表绑死全天行程。这三条标准必须同时满足，缺一不可。

不写的内容也要在序言里先讲清楚，以免读者在中途感到落差。本书不写穷游路线，青年旅舍、绿皮火车、十二小时硬座这些选项一概不在讨论范围之内，因为它们与本书的舒适优先预设直接冲突。本书不写自驾相关的路线安排，租车、停车场、川藏线山路这一类话题完全不会出现。本书也不写民宿和短租，原因在于这一类住宿的不确定性太高：临时取消、周边噪音、清洁状况这些因素，本书无法替读者把控。本书同样不写所谓的小众秘境，那些到了现场才发现没有维护、不通车或者基础设施缺失的地方，并不适合本书的目标读者，也不应该占据有限的假期。

剩下的部分就是正题。每一章的写作顺序大致相同：先告诉读者最佳的季节窗口与建议时长，再说明如何从杭州东抵达这座城市，最后展开住宿、景点、餐饮三个维度的具体推荐。结构尽量精简，决策尽量精准。希望这本书能在你计划下一次行程时，省下从头比较的时间。
