# 中国高铁游指南

杭州出发 · 不自驾 · 舒适优先 · 优先万豪 Bonvoy 体系。

这是一本写给从杭州东站出发、不开车、希望舒适胜过折腾、能用积分时优先动用万豪积分的读者的旅行手册。书中所有线路、所有车程、所有酒店推荐，都是从杭州东这一原点向外推演而成。

## 全部城市快速跳转

本书按照高铁车程从短到长划分为四个同心圈。每一座入选城市都按统一的格式展开：高铁班次与座位选择、万豪 Bonvoy 体系内的酒店清单（含位置、适合谁、价位、特色、餐饮）、当地具有调性的特色酒店、半天到三天的行程模板、景点详介、本地饮食推荐与最佳出行季节。

## 每个城市的标准格式

- 高铁班次与座位选择（在适用的情况下额外标注动卧选项）
- 万豪 Bonvoy 体系酒店在该城市的完整清单：每家酒店附上位置、适合人群、价位区间、特色亮点与餐饮评价
- 当地特色酒店：不一定是最贵的，但需要具备独特调性，例如扬州迎宾馆、西湖国宾馆这一类
- 行程模板：分别给出半天、一天、两天、三天四档
- 景点详介：每一处景点不少于 200 字，包含历史背景、核心看点、最佳游览时段、需要避开的常见坑位
- 本地饮食推荐
- 全年最佳出行季节
