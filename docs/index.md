# 中国高铁游指南

杭州出发 · 不自驾 · 舒适 · 优先住万豪。

## 用法

按距离分圈：

| 圈层 | 时长 | 城市 |
|---|---|---|
| 短途 | 1-2 h | 上海 / 苏州 / 南京 / 绍兴 / 宁波 / 黄山 |
| 中途 | 3-4 h | 北京 / 武汉 / 厦门 / 福州 / 长沙 |
| 长途 | 5-7 h | 西安 / 成都 / 重庆 / 广州 / 深圳 / 桂林 |

每个城市：

- 高铁班次与座位选择
- 万豪系酒店推荐（位置 / 价位 / 适合谁）
- 行程：半天 / 一天 / 两天 / 三天
- 景点详介：每个 200 字以上，写历史 / 看点 / 适合时段 / 避雷
- 吃：本地几家经典
- 季节：什么月份去最舒服

[季节推荐](chapters/05-seasonal.md) 反过来按月份索引：什么时候去哪里最值。

[实用信息](chapters/06-practical.md) 是高铁购票 / 万豪积分 / 行李 / 接驳的工具页。
