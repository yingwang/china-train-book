## 每个城市的格式

- 高铁班次与座位选择（动卧选项 where applicable）
- 万豪 Bonvoy 系酒店全城清单（每家位置 / 适合谁 / 价位 / 特色 / 餐饮）
- 当地特色酒店（不必最贵，要有调性，比如扬州迎宾馆 / 西湖国宾馆这种）
- 行程：半天 / 一天 / 两天 / 三天
- 景点详介：每个 200 字以上（历史 / 看点 / 适合时段 / 避雷）
- 本地吃
- 最佳季节
